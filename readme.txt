== Description ==
"Ads After First Paragraph" is a Ads plugin that includes user Ads code to post or webpage after first paragraph. First paragraph is the most revenue generating and high impression place in a blog. This plugin also has another feature of adding sticky ads banner vertically both sides, right and left. Sticky banner is visible during complete user visit to your page or website.

<strong>Features</strong><br>

1. Easy to use with Google AdSense and Bing or yahoo Ads Code.

2. You can use any JavaScript or HTML codes in any of three location left sidebar, right sidebar and first paragraph of blog page.

3. You can use any Ad Networks' code.

4. High Potential earning from your Website or Blog.

== Installation ==
1. Download plugin and unzip it to your `/wp-content/plugins/` plugin directory.
2. After that go to your plugin manager and activate it.
3. After activation, Ads After First Paragraph one configuration link is shown in plugin manager and it is also shown in admin menu.
4. Go to Ads After First Paragraph and Copy your Ads code in Ads After First Paragraph.
